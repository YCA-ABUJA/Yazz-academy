# Yazz Communication Academy LMS + CMS Platform

A comprehensive Learning Management System (LMS) and Content Management System (CMS) for Yazz Communication Academy.

## Features
- Multi-role access control (Admin, Teacher, Student, etc.)
- Course catalog management
- User registration with auto-generated usernames
- Learning tools (code sandbox, assignments, exams)
- Live lessons and chat
- Payment integration
- Certificate generation
- Reporting and analytics

## Technology Stack
- **Backend**: Python 3.9+, Flask 3.x
- **Database**: MySQL 8.0+
- **Frontend**: Bootstrap 5, Tailwind CSS, Alpine.js
- **Deployment**: Apache/nginx + uWSGI/Gunicorn
